# Movement Extraction

Extract constructive directions from PO provocations using de Bono's 4 movement types.

## Stages

### Stage 1: Movement Operation

Apply all 4 movement types to each provocation using movement-operation SOP:
- **Moment-to-moment**: Follow the provocation forward in time, what happens next?
- **Principle extraction**: What principle does this provocation embody?
- **Focus on difference**: What's different about the provoked world vs. current reality?
- **Positive aspects**: What's valuable or useful in the provocation?

### Stage 2: Stepping Stone

For directions that are interesting but impractical, use stepping-stone SOP to bridge toward practical solutions.

### Stage 3: Alternatives Generation

From the movement directions, generate concrete alternative approaches using alternatives-generation SOP.

## Minimum Yield

| Metric | Floor |
|--------|-------|
| Constructive directions from provocations | ≥4 |
| Movement types applied per provocation | all 4 |
| Stepping stones developed | ≥2 |
| Practical alternatives generated | ≥3 |

## Available SOPs

| SOP | Role |
|-----|------|
| movement-operation | Stage 1 — apply 4 movement types |
| stepping-stone | Stage 2 — bridge impractical to practical |
| alternatives-generation | Stage 3 — generate concrete alternatives |
