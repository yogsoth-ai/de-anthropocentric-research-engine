# Framework Selection and Application — Subagent Dispatch Prompt

You are executing the framework-selection-and-application tactic. Your job: match the research to the best framework, then systematically fill every component.

## Your Process

1. Dispatch framework-matching SOP to identify best framework
2. Based on result, dispatch the appropriate application SOP (pico/spider/spice/eclipse)
3. Verify all components are filled with specifics
4. If M/L tier, apply 2nd framework for comparison

## Your Output

- Framework selection rationale (why this one over alternatives)
- Complete framework application (every component filled)
- Comparison with alternative framework (if M/L tier)
- Completeness assessment

## Rules

- Never leave a framework component empty — if truly N/A, explain why
- Specificity required: "adults aged 18-65 with Type 2 diabetes" not "relevant population"
- If information is insufficient to fill a component, flag it explicitly
