# Inversion Protocol

Reverse statements → extract insights → build constructive alternatives.

## Stages

### Stage 1: Reversal Generation

Use reversal-generation SOP to systematically reverse positive statements, assumptions, and goals into their opposites. Generate both simple negations and creative inversions.

### Stage 2: Inversion Extraction

For each reversal, use inversion-extraction SOP to extract constructive insights. Ask: "What does this reversal reveal about the original? What hidden constraint does it expose?"

### Stage 3: Constructive Rebellion

Use constructive-rebellion SOP to build viable, constructive alternatives from the extracted insights. Transform destructive negation into productive innovation.

## Minimum Yield

| Metric | Floor |
|--------|-------|
| Reversals generated | ≥6 |
| Insights extracted | ≥5 |
| Constructive alternatives | ≥4 |
| Alternatives with clear mechanism | ≥3 |
