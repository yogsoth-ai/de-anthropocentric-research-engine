# Constraint Protocol

Inject constraints → force creative response → extract transferable principles.

## Stages

### Stage 1: Constraint Injection

Generate extreme constraints across multiple types (resource, time, scale, audience, material, inversion, context). Each constraint should make the current approach impossible, forcing genuinely new thinking.

### Stage 2: Constraint Response

For each constraint, generate a creative solution that satisfies both the original problem AND the constraint. No "impossible" — find a way. Use constraint-response SOP.

### Stage 3: Extract Transferable Principles

From each constrained solution, extract the underlying principle that made it work. Identify which principles apply even WITHOUT the constraint — these are the creative gold.

## Minimum Yield

| Metric | Floor |
|--------|-------|
| Constraints injected | ≥5 |
| Constraint-forced solutions | ≥3 |
| Transferable principles extracted | ≥3 |
| Principles validated as novel | ≥2 |

## Available SOPs

| SOP | Role |
|-----|------|
| constraint-response | Stage 2 — generate solutions under constraints |
| perspective-synthesis | Stage 3 — synthesize principles into report |
| novelty-scoring | Post — score novelty of extracted principles |
