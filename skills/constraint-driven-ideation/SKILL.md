## Execution Guidance

1. **Generate constraints**: Create 5-7 extreme constraints across types (resource, time, scale, audience, material, inversion, context)
2. **Apply sequentially**: Force a solution for each constraint — no "impossible" allowed
3. **Extract principles**: From each constrained solution, identify the transferable principle
4. **Combine**: Look for principles that reinforce each other
5. **Synthesize**: Build unconstrained solutions that incorporate the best constrained insights
