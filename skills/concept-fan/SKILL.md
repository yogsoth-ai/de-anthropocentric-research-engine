## Execution Guidance

1. **Define purpose**: Clarify the ultimate purpose (rightmost point of fan)
2. **Identify concepts**: Find broad approaches that serve the purpose
3. **Generate directions**: For each concept, identify specific directions
4. **Produce ideas**: For each direction, generate concrete ideas
5. **Escape and expand**: Use escape-technique if stuck at any level
6. **Synthesize**: Produce structured output via lateral-synthesis
