## Execution Guidance

1. **Select random word**: Choose genuinely random, concrete noun
2. **Extract attributes**: List properties, functions, associations of the random word
3. **Force connections**: Connect each attribute to the problem domain
4. **Movement**: Apply movement-operation to extract useful directions
5. **Develop**: Use stepping-stone to bridge impractical connections to practical ideas
6. **Synthesize**: Produce structured output via lateral-synthesis
