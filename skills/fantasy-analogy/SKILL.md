## Execution Guidance

1. **Suspend constraints**: Explicitly set aside all physical/economic/technical limits
2. **Wish**: Describe the perfect solution as if magic existed
3. **Analyze the wish**: What function does the fantasy fulfill?
4. **Approximate**: What real mechanisms come closest to the fantasy?
5. **Bridge**: Design feasible solutions that capture the fantasy's essence
6. **Validate**: Ensure the bridge solutions address the original need
