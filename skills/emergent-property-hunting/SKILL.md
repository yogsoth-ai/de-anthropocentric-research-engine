# Emergent Property Hunting

Seek properties that emerge from combination (non-additive). Focus specifically on identifying and validating properties that exist in the combination but not in any individual component — the hallmark of genuine creative novelty.

## Execution Guidance

1. **Generate Combinations**: Use combination-mapping to produce candidate combinations
2. **Predict Properties**: For each combination, predict what properties should exist if combination is merely additive
3. **Simulate**: Use blend-elaboration to run the combination as a mental simulation
4. **Detect Emergence**: Identify properties that appear in simulation but weren't predicted from additive model
5. **Validate**: Confirm emergent properties are genuine (not just overlooked additive properties)
6. **Characterize**: Describe the mechanism by which emergence occurs (interaction effects, threshold effects, structural effects)
7. **Rank**: Prioritize emergent properties by novelty and utility
