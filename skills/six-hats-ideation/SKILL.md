## Execution Guidance

1. **White Hat**: Gather facts and information about the problem
2. **Green Hat**: Run green-hat-session SOP — pure creative generation, no judgment
3. **Yellow Hat**: Find value in Green Hat ideas (positive assessment)
4. **Black Hat**: Identify risks and difficulties (deferred, not during generation)
5. **Red Hat**: Gut feelings about which ideas resonate
6. **Blue Hat**: Organize and select — process control
7. **Synthesize**: Produce structured output via lateral-synthesis
