# Provocation Generation

Generate PO provocations and extract constructive movement.

## Stages

### Stage 1: Assumption Surfacing

Identify the dominant assumptions and patterns in the current thinking using assumption-surfacing SOP. Prioritize paradigmatic assumptions with high disruption potential.

### Stage 2: Provocation Creation

Apply PO operators (escape, reversal, exaggeration, distortion, wishful thinking) to the surfaced assumptions using po-provocation SOP. Generate ≥10 provocations across all 5 types.

### Stage 3: Movement Extraction

For each promising provocation, extract constructive movement using movement-operation SOP (if available) or direct reasoning. Four movement types:
- Moment-to-moment: follow the provocation step by step
- Principle extraction: what principle does this suggest?
- Focus: what aspect does this highlight?
- Positive: what's the positive in this absurdity?

### Stage 4: Idea Formation

Convert movement directions into concrete ideas using idea-synthesis SOP. Each idea must have a clear mechanism, not just a vague direction.

## Minimum Yield

| Metric | Floor |
|--------|-------|
| Assumptions surfaced | ≥5 |
| Provocations generated | ≥10 |
| Movement directions extracted | ≥6 |
| Concrete ideas formed | ≥4 |

## Available SOPs

| SOP | Role |
|-----|------|
| assumption-surfacing | Stage 1 — enumerate assumptions |
| po-provocation | Stage 2 — generate PO statements |
| random-word-stimulus | Stage 2 — alternative stimulus source |
| movement-operation | Stage 3 — extract movement (if available) |
| idea-synthesis | Stage 4 — form concrete ideas |
| novelty-scoring | Post — score resulting ideas |
| saturation-detection | Post — check if more provocations needed |
